The Prediction Pipeline

Hamara system 3 basic steps mein kaam karta hai:

1. Data Intake

User se Age, BP, aur Cholesterol jaise parameters collect kiye jaate hain.

2. AI Analysis

Python backend data ko analyze karke risk patterns identify karta hai.

3. Risk Score

Model final probability score (High/Low) generate karta hai.
